Реализованы все фракталы.
При больших значениях рекурсии - программа может немножко заедать, но если нет сообщения,
что значение рекурсии для данного фрактала слишком велико - фрактал обязательно нарисуется!
Так же есть класс "Точка", который помогает в отрисовке фракталов, в этом классе в основном перегрузки операторов.
При изменении любого параметра - фрактал перерисуется.
Если какое-то из значений введено некорректно - оно автоматически заменяется на значение по умолчанию.

Максимальная глубина рекурсии для каждого из фракталов:
Фрактальное дерево - 14
Кривая Коха - 7
Ковёр Серпинского - 5
Треугольник Серпинского - 7
Множество Кантора - 9

Так же каждый фрактал реализован в своём классе, который, в свою очередь унаследован от базового класса "Fractal".
