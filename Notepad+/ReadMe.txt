В меню есть такая инструкция:
Настроить цетовую тему и автосохрание (в секундах) можно в настройках.
Настороить параметры всего текста или выделенного фрагмента можно в ячейке "Формат".
Автосохрание происходит сразу для всех вкладок в окне.
Причем после вызова автосохранения, все файлы сохраняются один раз.
(это сделано для того, чтобы сразу определить пути по которым эти файлы сохранять).
Горячие клавиши:
Ctrl + S - Сохранение текущего документа.
Ctrl + Shift + S - Сохранение всех открытых в окне документов.
Ctrl + N - Создание документа в новой вкладке.
Ctrl + Shift +  N - Создание документа в новом окне.
Ctrl + D - Закрытие вкладки.
Ctrl + Z - Закрытие приложения.
Логика кода такова:
Есть список вкладок и списки с текстами вкладок и путями к файлам для вкладок, всё работает через эти списки.
Так как начальный текста у меня хранятся в string есть еще bool переменные, которые показывают изменение файла
(тк string не видит форматирование).
rtf сохраняет форматирование.
Если пользователь хочет сохранить файл, но не указан путь для сохранение - автоматически вызывается функция "Сохранить как".
Если пользователь закрывает все вкладки, то открывается новая пустая вкладка.
Если пользователь хочет изменить формат текста и при этом ничего не выделил,
то форматирование применяется ко всему тексту и сохраняется при дальнейшей работе (как в настоящем блокноте!).
Перед закрытием приложения сохраняется цветовая тема, значение автосохранения и открытые файлы.
После перезапуска, если существуют дефолтные настройки - они открываются.
